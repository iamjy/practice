NAME

ApplicationShell - application shell widget from the X Toolkit  

SYNOPSIS

#include <X11/Shell.h>  

DESCRIPTION

ApplicationShell is a widget which is defined in the X Toolkit. We document its existence here for completeness. Please refer to the X Window System documentation for the real information regarding this widget.
