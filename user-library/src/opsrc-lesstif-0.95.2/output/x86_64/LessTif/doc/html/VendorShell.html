NAME

VendorShell - replaces VendorShell from the X toolkit  

SYNOPSIS

#include <X11/Shell.h>  

DESCRIPTION

VendorShell is a widget which is defined in the X Toolkit. When compiling and linking applications using e.g. the Athena Widget set, the X Toolkit VendorShell widget is used.

In applications using Motif or LessTif, an implementation of VendorShell that is specific to Motif or LessTif is used instead.

Therefore we do not document the X Toolkit VendorShell widget here.
